# @cool-ui/arkui

HarmonyOS 6 ArkUI components distributed as a HAR/ohpm package.

```ts
import { CoolButton, CoolComponentConfig } from '@cool-ui/arkui'

CoolButton({ config: new CoolComponentConfig('Continue') })
```

Open the package in the latest non-beta HarmonyOS 6 DevEco Studio and run `hvigorw assembleHar --mode module -p product=default` for the publishable HAR.

## Isolated npm consumer

A new temporary directory mirrored only `examples/npm-consumer`, `contracts/release.json`, and the two generated tgz files. It used a temporary `node_modules`; the directory was removed after the test. No public registry was needed because both dependencies were local `file:` tarballs.

```powershell
pnpm --dir $consumer install --offline --no-frozen-lockfile --lockfile-only --config.node-linker=hoisted
pnpm --dir $consumer install --offline --frozen-lockfile --config.node-linker=hoisted
pnpm --dir $consumer test
```

Result: PASS. The installed consumer resolved `@cool-ui/tokens@0.2.0` and `@cool-ui/wechat@0.2.0`, then verified canonical tokens, the 43-component manifest, MonthCalendar root/default generics, native Button WXML, and controlled TabBar/SegmentedControl source.

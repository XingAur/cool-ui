# NavigationRail

NavigationRail is a navigation component. Its contract aligns geometry and semantics while each implementation preserves platform-native behavior.

## API matrix

| SwiftUI | Compose | ArkUI | WeChat |
| --- | --- | --- | --- |
| `CoolNavigationRail` | `CoolNavigationRail` | `CoolNavigationRail` | `<cool-navigation-rail>` |

Maturity: SwiftUI **planned**, Compose **planned**, ArkUI **planned**, WeChat **planned**.
